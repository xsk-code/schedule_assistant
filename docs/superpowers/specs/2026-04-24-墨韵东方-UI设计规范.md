# 别蛮干 · 墨韵东方 UI 设计规范

> 版本：v1.0
> 日期：2026-04-24
> 设计方向：墨韵东方（Ink & Zen）
> 参考：每日故宫、中国传统色、水墨美学

---

## 一、设计理念

### 1.1 核心关键词

**墨韵 · 留白 · 朱印 · 宣纸**

紫微斗数是东方玄学，UI 应传达"翻开一卷古书"的体验，而非"使用一个工具"。

| 原则 | 说明 | 与旧版差异 |
|------|------|-----------|
| **墨韵而非平白** | 宣纸微纹理替代纯白底，墨色层次替代灰色梯度 | 旧版 `#FAFAF9` 纯暖白，缺乏质感 |
| **留白而非填满** | 大量呼吸空间，内容区如卷轴般展开 | 旧版间距单调，视觉节奏平 |
| **朱印而非色块** | 品牌色如朱砂印章般点缀，而非大面积铺色 | 旧版 `#D97757` 大面积按钮，缺乏文化感 |
| **毛笔而非线条** | 色条如毛笔笔触，有粗细变化和飞白感 | 旧版 2rpx 直线色条，过于机械 |
| **卷轴而非卡片** | 信息如卷轴般纵向展开，靠留白和淡墨线分隔 | 旧版白卡片叠暖白底，层级模糊 |

### 1.2 视觉隐喻

```
用户打开小程序 = 翻开一卷古书
四化信息展示   = 展开一幅命理卷轴
任务输入       = 在宣纸上落笔
分析结果       = 命理师的朱批笺注
行动步骤       = 朱笔圈点的待办清单
```

## 二、配色系统

### 2.1 主色板

基于中国传统色，降低饱和度以适配屏幕显示（原则：主色占比 ≤ 40%，辅助色需有文化出处）。

```
页面底色（宣纸）：
  --color-bg:           #F5F0E8    宣纸色（暖米白，非纯白）
  --color-bg-warm:      #EDE7DB    陈年宣纸（深一阶，用于输入区底色）

墨色梯度（焦浓重淡清）：
  --color-ink-1:        #2C2420    焦墨（主文字，比纯黑暖）
  --color-ink-2:        #5C4F45    浓墨（次文字）
  --color-ink-3:        #8C7E72    重墨（辅助文字）
  --color-ink-4:        #B5A99A    淡墨（占位文字、淡色标签）
  --color-ink-5:        #D8CFC3    清墨（分割线、极淡装饰）

品牌色（朱砂系）：
  --color-vermilion:    #C4463A    朱砂（品牌主色，取代旧版 Terra Cotta）
  --color-vermilion-light: #F2E0DC  朱砂浅底（hover / 浅底标签）
  --color-vermilion-dark:  #9E352C  朱砂深色（pressed 状态）
  --color-seal:         #B83A2E    印泥色（印章装饰用，比朱砂更深）

四化能量色（传统矿物色，降低饱和度）：
  --color-lu:           #5E8B6A    石绿（禄，柔绿，取自石绿矿物）
  --color-quan:         #B8944C    藤黄（权，暖金，取自藤黄颜料）
  --color-ke:           #6B8FAB    花青（科，雾蓝，取自花青颜料）
  --color-ji:           #B46A5E    赭石（忌，柔红，取自赭石矿物）

交互色：
  --color-primary:      #C4463A    主按钮（朱砂色）
  --color-primary-text: #F5F0E8    主按钮文字（宣纸色）
```

### 2.2 配色策略变化

| 场景 | 旧版 | 墨韵东方 |
|------|------|---------|
| 页面底色 | `#FAFAF9` 纯暖白 | `#F5F0E8` 宣纸色（微黄，有质感） |
| 主文字 | `#1C1917` 暖黑 | `#2C2420` 焦墨（更暖，有墨韵） |
| 次文字 | `#78716C` 灰棕 | `#5C4F45` 浓墨（更深，对比更强） |
| 品牌色 | `#D97757` Terra Cotta | `#C4463A` 朱砂（更正红，更有文化感） |
| 四化色 | 现代感饱和色 | 传统矿物色（降低饱和度，更雅致） |
| 分割线 | `rgba(0,0,0,0.06)` | `#D8CFC3` 清墨色（更暖，更自然） |

### 2.3 墨色五阶

中国传统水墨有"焦浓重淡清"五阶，用于建立视觉层级：

```
焦墨 #2C2420  ████████  主标题、主按钮文字
浓墨 #5C4F45  ████████  正文、次标题
重墨 #8C7E72  ████████  辅助说明、标签
淡墨 #B5A99A  ████████  占位文字、淡色标签
清墨 #D8CFC3  ████████  分割线、极淡装饰
```

## 三、字体规范

### 3.1 字体栈

```
标题（Display）：  "Noto Serif SC", "Source Han Serif SC", "PingFang SC", serif
  · 紫微斗数属东方玄学，标题用宋体/衬线体传达文化感

正文（Body）：     "PingFang SC", -apple-system, sans-serif
  · 正文用无衬线体保证可读性（小字号衬线体在屏幕上可读性差）

数字/干支：       "Noto Serif SC", Georgia, serif
  · 天干地支、日期等用衬线体，增强古典感
```

> ⚠️ 微信小程序中自定义字体需要加载网络字体文件，会增加包体积和加载时间。
> 推荐方案：标题用系统 serif 字体栈（Georgia + PingFang SC），不额外加载字体。
> 如需更精致的宋体效果，可后续引入 "Noto Serif SC" 子集。

### 3.2 字号规范

```
标题层级（用字号跳跃建立层级，跳跃 ≥ 1.5x）：
  · 页面大标题：  52rpx   font-weight: 700   line-height: 1.2   letter-spacing: -0.02em
  · 区块标题：    36rpx   font-weight: 600   line-height: 1.3   letter-spacing: 0
  · 小节标题：    30rpx   font-weight: 600   line-height: 1.4

正文层级：
  · 主要内容：    28rpx   font-weight: 400   line-height: 1.8   （行高加大，增加呼吸感）
  · 辅助说明：    24rpx   font-weight: 400   line-height: 1.6
  · 标签/微标：   22rpx   font-weight: 500   letter-spacing: 0.04em

关键变化（vs 旧版）：
  · 页面大标题从 56rpx → 52rpx（稍小但用衬线体补偿视觉重量）
  · 正文行高从 1.75 → 1.8（更宽松，卷轴感）
  · 标题字体从 sans-serif → serif（文化感）
  · 标签 letter-spacing 从 0.02em → 0.04em（更疏朗）
```

## 四、间距系统

### 4.1 间距规范

```
基于 8rpx 网格：
  xs:   8rpx    组件内微间距
  sm:  16rpx    组件内间距
  md:  24rpx    组件间间距
  lg:  40rpx    区块间间距
  xl:  56rpx    大区块间距
  2xl: 80rpx    页面级间距
  3xl: 120rpx   卷轴展开间距（新增，用于大区块之间的"呼吸"）

关键变化（vs 旧版）：
  · 页面顶部留白从 80rpx → 96rpx（给标题更多"卷首"空间）
  · 区块间距从 40rpx → 56rpx（增加呼吸感）
  · 新增 3xl: 120rpx 用于首页四化区与输入区之间的大留白
  · 内容区左右 padding 从 32rpx → 40rpx（更宽敞）
```

## 五、组件设计规范

### 5.1 四化能量卷轴（SihuaBar → SihuaScroll）

**核心变化：从"信息条"到"命理卷轴"**

```
旧版：
  今日 甲子日
  │禄廉贞  │权破军  │科武曲  │忌太阳

新版：
  ┌─────────────────────────────────────┐
  │                                     │
  │  甲子  ·  谷雨后三日                │  ← 衬线体，干支+节气
  │                                     │
  │  ◆ 化禄廉贞   ◆ 化权破军           │  ◆ 为毛笔笔触圆点
  │  ◆ 化科武曲   ◆ 化忌太阳           │  各自带有对应矿物色
  │                                     │
  │  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  │  ← 淡墨虚线（卷轴分隔）
  │                                     │
  └─────────────────────────────────────┘
```

设计要点：
- 干支用衬线体大号显示，节气用淡墨小字
- 四化用 ◆ 圆点（8rpx）+ 衬线文字，不用左边框
- 底部用淡墨虚线分隔，暗示"卷轴展开"
- 整体无可见容器，靠留白和字重区分

### 5.2 任务输入区（TaskInput）

**核心变化：从"输入框"到"宣纸落笔区"**

```
旧版：
  ┌─────────────────────────┐
  │  暖白底 #FAFAF9          │  20rpx 圆角
  │  描述你想要完成的任务...  │
  └─────────────────────────┘
  [⚡] 简单    [🧠] 思考
  ┌─────────────────────────┐
  │    开始智能拆分           │  品牌色底
  └─────────────────────────┘

新版：
  ┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐
  │                          │  陈年宣纸底 #EDE7DB
  │  落笔写下今日所虑…       │  24rpx 圆角
  │                          │  淡墨虚线边框
  └─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘
  ○ 速览    ○ 深思           ← 空心圆点 + 文字，选中态实心朱砂
  ┌─────────────────────────┐
  │    落笔开卷    →         │  朱砂底 + 宣纸色字
  └─────────────────────────┘
```

设计要点：
- 输入区底色用"陈年宣纸" `#EDE7DB`，比页面底色深一阶
- 边框用淡墨虚线（`border: 2rpx dashed #D8CFC3`），暗示"书写区域"
- 模式切换：空心圆点 → 实心朱砂圆点（不用 pill 背景色块）
- 提交按钮：朱砂底 `#C4463A` + 宣纸色字 `#F5F0E8`
- placeholder 文案改为"落笔写下今日所虑…"

### 5.3 分析结果卡片（ResultCard）

**核心变化：从"信息堆叠"到"朱批笺注"**

```
新版布局：

  今日待办                            ← 36rpx 衬线体，浓墨
  ━━━━━━━━━━━━━━━━━━━━━━            ← 朱砂色短横线（装饰，如朱笔划重点）
  一句话总结任务核心                   ← 28rpx 无衬线，浓墨

  ○ 步骤1标题                         ← 30rpx 无衬线
    步骤描述文字...                    ← 26rpx 无衬线，重墨
    高优 · 30分钟                      ← 22rpx，淡墨

  ○ 步骤2标题
    ...

  ▾ 四化深解                          ← 折叠区，点击展开
    展开后：

    ◆ 化禄 · 廉贞                     ← 石绿圆点 + 衬线标题
    ┃ 化禄带来机遇...                  ← 左侧毛笔笔触色条（4rpx，有粗细变化）
    ┃                                  ← 衬线正文

    ◆ 化权 · 破军                     ← 藤黄圆点
    ┃ 化权赋予掌控力...

    ◆ 化科 · 武曲                     ← 花青圆点
    ┃ 化科利于学习...

    ◆ 化忌 · 太阳                     ← 赭石圆点
    ┃ 化忌需要注意...

  ━━━━━━━━━━━━━━━━━━━━━━            ← 淡墨分隔线

  ◈ 最佳切入点                        ← ◈ 为朱砂印章风格装饰
    原因说明...
    建议说明...

  ┃ 整体建议                          ← 左侧朱砂色条
  ┃ 综合分析文字...
```

设计要点：
- "今日待办"下方加朱砂色短横线装饰（如朱笔划重点）
- 四维分析默认折叠，点击"四化深解"展开
- 色条从 2rpx → 4rpx，模拟毛笔笔触（可用 border-left + border-radius 不规则模拟）
- 最佳切入点用 ◈ 印章风格装饰
- 整体建议用朱砂色条（品牌色），而非灰色条

### 5.4 对话面板（ConversationPanel）

**核心变化：从"聊天气泡"到"命理师问诊"**

```
新版布局：

  第 2 轮 / 共 3 轮                   ← 淡墨文字
  ○ ○ ● ○                            ← 进度点，当前为朱砂实心

  ┌─────────────────────────┐
  │  这个任务最让你头疼的是？ │  ← 宣纸底色 + 淡墨边框
  └─────────────────────────┘         ← 不是品牌色浅底，更素雅
  命理师想了解…                       ← 淡墨小字提示

  ○ 选项一                            ← 空心圆点 + 文字
  ○ 选项二                            ← hover: 朱砂实心圆点
  ○ 选项三

  跳过 →                              ← 淡墨文字链接
```

### 5.5 空状态（Empty）

**核心变化：从"emoji"到"水墨留白"**

```
旧版：
  📭
  输入您的任务
  别蛮干将根据今日四化能量…

新版：
  ┌─────────────────────────┐
  │                         │
  │     ╱                   │  ← 淡墨山水剪影（CSS/SVG）
  │   ╱   ╲                │     极简，3-4笔
  │  ╱     ╲   ╱           │
  │         ╲╱              │
  │                         │
  │  落笔写下所虑           │  ← 浓墨，衬线体
  │  四化能量将为你指引方向  │  ← 重墨，无衬线
  │                         │
  └─────────────────────────┘
```

设计要点：
- 去除 emoji，用淡墨山水剪影替代
- 山水剪影用 CSS 渐变或内联 SVG 实现（极简，3-4笔）
- 文案改为更古典的表达

### 5.6 加载状态（Loading）

**核心变化：从"旋转点"到"墨滴晕染"**

```
新版：
  ○ ○ ○                              ← 三个墨点依次从淡到浓
                                      ← 模拟墨滴落纸晕染的效果

  正在研读天机…                       ← 衬线体，淡墨
```

设计要点：
- 三个圆点从淡墨 → 浓墨依次变化
- 动画模拟墨滴晕染：scale(0.8) opacity:0.3 → scale(1.2) opacity:1 → scale(1) opacity:0.6
- 文案改为"正在研读天机…"

### 5.7 行动步骤（ActionStep）

**核心变化：从"待办清单"到"朱笔圈点"**

```
新版：
  ○ 步骤标题                           ← 空心圆，边框色=对应四化色
    步骤描述...                         ← 衬线正文
    高优 · 30分钟                       ← 淡墨标签

  完成态：
  ● 步骤标题                           ← 实心朱砂圆
    ~~步骤描述~~                        ← 删除线 + 降低透明度
```

设计要点：
- checkbox 圆形，边框色对应四化维度色
- 完成态用朱砂色实心圆（而非品牌色），如朱笔圈点
- 标签去除背景色块，纯淡墨文字

### 5.8 历史记录项（HistoryItem）

**核心变化：从"列表卡片"到"案卷条目"**

```
新版：
  任务描述文字                         ← 浓墨，无衬线
  四月廿三  甲子日                      ← 淡墨，衬线体

  ◆禄廉贞  ◆权破军  ◆科武曲  ◆忌太阳  ← 各自矿物色圆点

  今日待办
  一句话总结...                        ← 重墨，两行截断

  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─  ← 淡墨虚线分隔

## 六、页面设计

### 6.1 首页（index）

**布局原则：标题区居中，内容区左对齐。** 首屏如卷首题签，标题居中彰显庄重；下方内容如卷轴展开，左对齐便于阅读。

```
┌─────────────────────────────────┐
│  状态栏（导航栏色：#2C2420）      │
├─────────────────────────────────┤
│                                 │
│          别蛮干                  │  ← 居中，52rpx 衬线体 700，焦墨
│       找对方向再出手              │  ← 居中，24rpx 无衬线，淡墨
│                                 │
│          甲子日                  │  ← 居中，36rpx 衬线体 600，浓墨
│       谷雨后三日                 │  ← 居中，22rpx 衬线，淡墨
│                                 │
│   ◆ 化禄廉贞   ◆ 化权破军       │  ← 居中排列，26rpx 衬线，各矿物色
│   ◆ 化科武曲   ◆ 化忌太阳       │
│                                 │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   │  ← 淡墨虚线（卷轴分隔）
│                                 │
│  ┌─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┐   │
│  │                          │   │
│  │  落笔写下今日所虑…       │   │  ← 左对齐，陈年宣纸底，淡墨虚线边框
│  │                          │   │
│  └─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─┘   │
│                                 │
│  ○ 速览    ○ 深思               │  ← 左对齐，模式切换
│                                 │
│  ┌─────────────────────────┐   │
│  │     落笔开卷    →        │   │  ← 朱砂底 + 宣纸色字
│  └─────────────────────────┘   │
│                                 │
│       （空状态/分析结果区）       │  ← 左对齐
│                                 │
│                                 │
│  ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─   │
│     基于紫微斗数四化理论         │  ← 居中，页脚，淡墨
│                                 │
└─────────────────────────────────┘
```

**首页布局要点：**

| 区域 | 对齐方式 | 说明 |
|------|---------|------|
| 标题"别蛮干" | **居中** | 如卷首题签，庄重醒目 |
| 副标题"找对方向再出手" | **居中** | 与标题同组，居中呼应 |
| 干支日期 | **居中** | 今日天机，居中凸显 |
| 四化能量 | **居中** | 两行两列居中排列，如命盘对称 |
| 输入区及以下 | **左对齐** | 卷轴展开，阅读自然 |
| 页脚 | **居中** | 收束，与标题区首尾呼应 |

### 6.2 历史页（history）

```
┌─────────────────────────────────┐
│  状态栏                          │
├─────────────────────────────────┤
│                                 │
│  案卷                            │  ← 52rpx 衬线体 700
│  共 12 条                        │  ← 淡墨
